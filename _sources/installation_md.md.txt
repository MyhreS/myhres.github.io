# Installation
Installation is simple. Just use pip to install the package.

```bash
pip install easyaug
```

## Quick setup
To use Quickviewer:

```python
from easyaug.quickview import Quickviewer
quickviewer = Quickviewer()
```

To use Preprocesser:

```python
from easyaug.preprocess import Preprocesser
preprocesser = Preprocesser()
```

To use Augmenter:

```python
from easyaug.augment import Augmenter
augmenter = Augmenter()
```
